Papers Graph Databases
Design and load graph database with academical paper data using both GraphDB and Neo4J.
The goal of the project is creating a TBOX, the equivalent of schema definitions for knowledge graphs, creating an ABOX, the corresponding data instances, linking these together and finally querying it. The context considers academical papers and relevant concepts like reviews, journal, and conferences. For more information, visit my GitHub page.
